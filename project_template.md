# Tradewind — Maritime Shipping Analytics Platform
> Stack: React · TypeScript · Node/Express · PostgreSQL

---

## What Tradewind Is

A web-based maritime analytics platform that ingests real-time AIS vessel tracking data and trade flow data, stores it in PostgreSQL, and presents it through a React dashboard. Target users: small freight brokers, logistics SMEs, and port agents who need maritime intelligence but can't afford enterprise tools.
